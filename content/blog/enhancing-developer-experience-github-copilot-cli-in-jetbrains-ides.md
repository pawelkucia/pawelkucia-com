## Introducing Copilot CLI in JetBrains IDEs

GitHub has expanded its Copilot experience by integrating Copilot CLI directly into JetBrains IDEs. This update is designed to significantly improve developer workflows by enabling advanced command-line interface (CLI) session capabilities.

## What’s New?

The main focus of this update is the introduction of agentic improvements. These enhancements make Copilot smarter in handling tasks autonomously within CLI sessions, potentially reducing repetitive work and speeding up coding processes.

## Why It Matters

By combining GitHub Copilot’s AI-driven suggestions with the power of JetBrains environments, developers can achieve greater productivity. This integration helps in maintaining flow without switching contexts and supports more natural interaction with coding tasks.

## Looking Forward

As AI technologies continue to embed deeper into development tools, this integration marks an important step toward more autonomous and efficient programming workflows. Developers should evaluate how these capabilities can best fit their routines to leverage the full potential of AI assistance.
