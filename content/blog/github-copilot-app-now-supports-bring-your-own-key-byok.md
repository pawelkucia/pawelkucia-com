## Introduction
GitHub Copilot, the AI-powered coding assistant, has recently expanded its capabilities with support for Bring Your Own Key (BYOK). This update allows developers to run agent sessions using their own preferred AI model providers.

## What is BYOK Support?
BYOK means you can connect GitHub Copilot to various model providers such as OpenAI, Azure OpenAI, Microsoft Foundry, Anthropic, and others. This setup provides enhanced security and flexibility by giving users control over encryption keys and models.

## Benefits for Developers
With BYOK support, developers can integrate GitHub Copilot seamlessly into their existing AI infrastructure. It enables better compliance and governance while maintaining the convenience of AI-powered coding recommendations.

## Conclusion
The addition of BYOK support marks a significant step toward customizable and secure AI developer tools, empowering teams to tailor their AI environment according to organizational needs.
