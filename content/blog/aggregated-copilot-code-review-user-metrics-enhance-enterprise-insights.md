## Enhanced Visibility into Copilot Usage

GitHub recently updated its Copilot usage metrics API to include aggregated counts of both active and passive users involved in Copilot code reviews. This enhancement is particularly beneficial for enterprises and organizations seeking a holistic view of their team's engagement.

## Understanding Active vs. Passive Users

Active users contribute directly to code reviews using Copilot, while passive users may be recipients or observers within the workflow. By aggregating these metrics, organizations can better assess adoption, productivity, and collaboration patterns.

## Why This Matters

Improved metrics facilitate more informed decisions around training, resource allocation, and measuring the impact of AI-assisted development tools within teams. This advancement signifies GitHub's commitment to evolving its tools in alignment with enterprise needs.
