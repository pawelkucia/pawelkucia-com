## Expanding the GitHub Copilot CLI Experience: Introducing Remote Sessions

GitHub Copilot CLI is transforming how developers interact with command-line tools. Traditionally confined to local machines, it now offers remote control capabilities through the new `copilot --remote` feature.

## What Does copilot --remote Enable?

With this update, you can monitor and manage running CLI sessions directly from web or mobile devices. This removes the constraints of having to be physically at the machine running the command, allowing for greater flexibility and responsiveness in development workflows.

## Benefits for Developers

- **Increased Mobility:** Control your CLI sessions from any device connected to the internet.
- **Enhanced Productivity:** Quickly address issues or steer processes without being tied down.
- **Streamlined Collaboration:** Easily share CLI session states when troubleshooting remotely.

This public preview marks a significant step in making command-line interfaces more accessible and adaptable for modern development needs.
