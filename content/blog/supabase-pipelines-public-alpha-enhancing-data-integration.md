## Supabase Pipelines Enter Public Alpha

Supabase Pipelines is now in public alpha, introducing several meaningful updates for data professionals. This release focuses on improving reliability and speed in managing data workflows.

## Key Features: Schema Change Support and Faster Sync

One major enhancement is the support for schema changes, allowing pipelines to adapt dynamically as databases evolve. Additionally, the initial synchronization has been optimized for greater speed, reducing wait times when setting up data flows.

## Expanded Destination Support

The update also brings a new destination request form that includes popular analytics databases such as ClickHouse, Snowflake, and DuckLake. This enables users to route data efficiently to these platforms, facilitating advanced analytics and data warehousing.

## Conclusion

These additions mark a significant step toward robust and flexible data pipeline management within Supabase, balancing ease of use with powerful integration capabilities.
