## Broadening the Rubber Duck's Capabilities

GitHub Copilot CLI's Rubber Duck review agent, known for its efficient cross-family review capabilities, has taken a significant step in augmenting its flexibility. Previously available for GPT-based sessions, it now integrates Claude-powered critic agents, widening the scope of AI models it supports.

## What This Means for Developers

By supporting Claude in addition to GPT models, Rubber Duck facilitates a more versatile review environment. Developers using GitHub Copilot CLI can now experience nuanced code critiques powered by different AI architectures during their coding sessions.

## Enhancing AI-Driven Code Reviews

This development aligns with the growing trend of multi-model AI collaboration, offering enriched insights and diversified perspectives within automated code review tools. It demonstrates advancements toward more robust AI integration in developer workflows.
