## Setting Standards with Enterprise-Managed Plugins

GitHub has launched a feature that enables enterprise administrators to configure and distribute plugins to GitHub Copilot CLI users across their entire organization. This advancement helps enforce baseline standards and ensure consistency in development environments.

## How This Benefits Enterprises

The ability to centrally manage plugins means organizations can enhance security by controlling what tools are used, streamline workflows by making key plugins readily available, and maintain compliance with internal policies.

## Public Preview Availability

Currently in public preview, the feature invites enterprises to experience improved management of Copilot CLI plugins. This can lead to smoother collaboration and greater efficiency for teams leveraging AI-assisted coding.

The update reflects GitHub's ongoing commitment to expanding enterprise capabilities and empowering developers with secure and standardized tools.
